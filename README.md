## 🧑‍💻 Using Each Tool

### Beginner Walkthrough  
Open this to see simple steps that explain the basics of tissue culture and hydroponics. It includes pictures and plain instructions.

### SOPs  
Select any of the 10 SOPs to get clear methods. These cover preparing media, sterilization, propagation, and nutrient supply.

### Sterilization Calculator  
Input parameters like volume, container size, and chemical strength. The tool then shows how long to sterilize for safety.

### Media Formulations  
Use this to create exact recipes for your growth solutions. Add ingredients by weight or volume, and the calculator totals the mixture.

### Growth Regulators  
Track and adjust levels of hormones like cytokinins and auxins to control plant growth phases.

### EC/pH Tools  
Measure your water’s electrical conductivity and pH values, then enter them into the tool. It recommends adjustments needed for ideal plant health.
